# CRM системы

Системы управления взаимоотношениями с клиентами (Customer Relationship Management).

## Что такое CRM?

CRM система помогает управлять:
- Клиентами и их контактами
- Проектами и задачами
- Продажами и сделками
- Взаимодействием с клиентами

## Примеры

- [Веб-студия](./webstudio.md) - Управление клиентами и проектами для веб-студии
- [Кофейня](./coffee-shop.md) - Управление заказами и клиентами для кофейни

## Общие сущности

Для любой CRM системы обычно нужны:

1. **Client (Клиент)** - Информация о клиенте
2. **Project (Проект)** - Проекты/сделки клиента
3. **Task (Задача)** - Задачи по проектам
4. **Contact (Контакт)** - Контактные лица клиента
5. **Deal (Сделка)** - Продажи и сделки

## Архитектура

Все CRM примеры используют базовую архитектуру проекта:

```
Domain (Client, Project, Deal)
    ↓
Repository (ClientRepository, ProjectRepository)
    ↓
Use Case (ClientUseCase, ProjectUseCase)
    ↓
Handler (ClientHandler, ProjectHandler)
    ↓
Router (/api/v1/clients, /api/v1/projects)
```
