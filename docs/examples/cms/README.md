# CMS системы

Системы управления контентом (Content Management System).

## Что такое CMS?

CMS система помогает управлять контентом веб-сайта:
- Статьями и страницами
- Медиа файлами
- Категориями и тегами
- Комментариями

## Примеры

- [Блог](./blog.md) - Простой блог как WordPress
- [Мебельный магазин](./furniture-store.md) - Интернет-магазин мебели

## Общие сущности

Для любой CMS системы обычно нужны:

1. **Post/Article (Статья)** - Основной контент
2. **Category (Категория)** - Организация контента
3. **Tag (Тег)** - Метки для поиска
4. **Media (Медиа)** - Изображения и файлы
5. **Comment (Комментарий)** - Обратная связь от пользователей

## Особенности CMS

### SEO оптимизация

- ЧПУ (slug) для статей
- Мета-теги
- Структурированные данные

### Мультиязычность

- Поддержка нескольких языков
- Переводы контента

### Версионирование

- История изменений
- Возможность отката
