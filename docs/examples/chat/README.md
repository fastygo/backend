# Системы чатов

Системы обмена сообщениями в реальном времени.

## Что такое чат система?

Чат система позволяет:
- Отправлять и получать сообщения
- Создавать комнаты/каналы
- Управлять пользователями и ролями
- Поддерживать историю сообщений

## Примеры

- [Простой чат](./simple.md) - Базовый чат без ролей
- [Чат с ролями](./with-roles.md) - Чат с системой ролей и агентами поддержки

## Общие сущности

Для любой чат системы обычно нужны:

1. **Room/Channel (Комната)** - Где происходит общение
2. **Message (Сообщение)** - Само сообщение
3. **Member (Участник)** - Участник комнаты
4. **Role (Роль)** - Роль участника (admin, member, agent)

## Особенности чатов

### Real-time обновления

- WebSocket соединения
- Push уведомления
- Онлайн статусы

### История сообщений

- Хранение всех сообщений
- Поиск по истории
- Пагинация

### Модерация

- Удаление сообщений
- Блокировка пользователей
- Фильтрация контента
