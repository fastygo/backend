# Развертывание

Руководство по развертыванию приложения на различных платформах.

## Содержание

- [Dokploy](./dokploy.md) - Развертывание на Dokploy (облачный сервер)
- [Docker](./docker.md) - Развертывание через Docker
- [Production Checklist](./production-checklist.md) - Чеклист для продакшена

## Общие принципы

### 1. Безопасность

- Используйте сильные пароли
- Настройте HTTPS
- Ограничьте доступ к БД
- Используйте секреты для конфигурации

### 2. Мониторинг

- Настройте логирование
- Добавьте метрики
- Настройте алерты
- Мониторьте здоровье сервиса

### 3. Масштабирование

- Используйте load balancer
- Настройте репликацию БД
- Используйте Redis cluster
- Настройте горизонтальное масштабирование

## Быстрый старт

### Локальное развертывание (Docker Compose)

```bash
docker-compose up -d
```

### Production развертывание

1. Соберите Docker образ
2. Настройте переменные окружения
3. Запустите контейнеры
4. Настройте reverse proxy
